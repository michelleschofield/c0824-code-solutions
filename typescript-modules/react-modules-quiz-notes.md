## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What are modules?
  separate files that can import and export things and are great for organization
- What is the difference between named and default imports?
  default imports can be named whatever you want when you import them while named have to be the same name it was defined in the module it's from. Named imports have to be in curly brackets
- What is the difference between named and default exports?
  there can only be one default export per module and there can be as many named as you want and they have to be imported differently. Default exports use the default keyword
